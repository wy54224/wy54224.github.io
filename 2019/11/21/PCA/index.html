从PCA到概率图模型做Multi-view Outlier Detection

发表于 2019-11-21 分类于 maching learning

1. Principal Component Analysis(PCA)

1.1 问题分析

主成分分析——寻找一个超平面，尽量将所有样本正确表达。 1. 最近重构性：样本点到这个超平面的目标足够近； 2. 最大可分性：略。

从最近重构性的角度来说，PCA的目标函数可以构造如下：

\[ \begin{aligned} \min\limits_{\mathbf{A}, \mathbf{E}}&\quad\|\mathbf{E}\|_F\\ \text{subject to}&\quad \mathop{rank}(\mathbf{A})\leq r, \mathbf{D} = \mathbf{A} + \mathbf{E} \end{aligned} \]

1.2 优化

假设投影的低维超平面由\[r\]个正交的单位坐标\[\mathbf{W} = \{\mathbf{w}_1, \mathbf{w}_2, ..., \mathbf{w}_r\}\in\mathbb{R}^{d\times r}\]构成，那么存在投影关系\(\mathbf{a}_i=\mathbf{W}\mathbf{z}_i\)，\(\mathbf{z}_i\in\mathbb{R}^r\)是\(\mathbf{d}_i\)在超平面\(\mathbf{W}\)上的投影。而\(\mathbf{z}_i\)又可以通过\(\mathbf{d}_i\)和\(\mathbf{W}\)的列向量逐一点积得到\(\mathbf{z}_i = \mathbf{W}^T\mathbf{d}_i\)，有\(\mathbf{A} = \mathbf{W}\mathbf{W}^T\mathbf{D}\)。退而求其次，如果优化error项的平方，优化目标可以变为： \[ \begin{aligned} \|\mathbf{E}\|_F^2 &= \|\mathbf{D} - \mathbf{A}\|_F^2\\ &= \mathop{Tr}[(\mathbf{D} - \mathbf{A})^T(\mathbf{D} - \mathbf{A})]\\ &= \mathop{Tr}(\mathbf{D}^T\mathbf{D} - 2\mathbf{A}^T\mathbf{D} + \mathbf{A}^T\mathbf{A})\\ &= \mathop{Tr}(\mathbf{D}^T\mathbf{D} - 2\mathbf{D}^T\mathbf{W}\mathbf{W}^T\mathbf{D} + \mathbf{D}^T\mathbf{W}\mathbf{W}^T\mathbf{W}\mathbf{W}^T\mathbf{D})\\ &= \mathop{Tr}(\mathbf{D}^T\mathbf{D} - 2\mathbf{D}^T\mathbf{W}\mathbf{W}^T\mathbf{D} + \mathbf{D}^T\mathbf{W}\mathbf{\mathrm{I}}\mathbf{W}^T\mathbf{D})\\ &= \mathop{Tr}(\mathbf{D}^T\mathbf{D} - \mathbf{D}^T\mathbf{W}\mathbf{W}^T\mathbf{D}) \end{aligned} \] 因此，最终优化的目标函数为： \[ \begin{aligned} \max\limits_{\mathbf{W}}&\quad\mathop{Tr}(\mathbf{W}^T\mathbf{D}\mathbf{D}^T\mathbf{W})\\ \text{subject to}&\quad\mathbf{W}^T\mathbf{W} = \mathbf{\mathrm{I}} \end{aligned} \] 这是一个很典型的谱分解求解的问题，套一下Ky Fan's Theorem直接SVD分解就好了。

2.Robust Principal Component Analysis(RPCA)

2.1 问题分析

加上子空间学习常用的策略——低秩+稀疏： \[ \begin{aligned} \min\limits_{\mathbf{A}, \mathbf{E}}&\quad\|\mathbf{A}\|_* + \lambda\|\mathbf{E}\|_1\\ \text{subject to}&\quad\mathbf{D} = \mathbf{A} + \mathbf{E} \end{aligned} \] ### 2.2 优化 带等式约束的问题，直接用梯度上升法最大化对偶问题，增广拉格朗日函数为： \[ \mathop{L}(\mathbf{A}, \mathbf{E}, \mathbf{Y}, \mathbf{\mu}) = \|\mathbf{A}\|_* + \lambda\|\mathbf{E}\|_1 + <\mathbf{Y}, \mathbf{D} - \mathbf{A} - \mathbf{E}> + \frac{\mu}{2}\|\mathbf{D} - \mathbf{A} - \mathbf{E}\|_F^2 \] 定义soft-thresholding(shrinkage)函数： \[ \mathop{\mathcal{S}_{\epsilon}}[x] = \begin{cases} x - \epsilon,&\text{若}x > \epsilon\\ x + \epsilon,&\text{若}x < -\epsilon\\ 0,&\text{其他} \end{cases} \] 其中\(x\in\mathbb{R}\)和\(\epsilon>0\)，可以将这个操作扩展到矩阵上，对矩阵逐元素操作。有下面的结论： \[ \begin{aligned} \mathbf{U}\mathop{\mathcal{S}_{\epsilon}}[\mathbf{S}]\mathbf{V}^T &= \mathop{\mathrm{argmin}}\limits_{\mathbf{X}}\;\epsilon\|\mathbf{X}\|_*+\frac{1}{2}\|\mathbf{X} - \mathbf{W}\|_F^2\\ \mathop{\mathcal{S}_{\epsilon}}[\mathbf{W}] &= \mathop{\mathrm{argmin}}\limits_{\mathbf{X}}\;\epsilon\|\mathbf{X}\|_1+\frac{1}{2}\|\mathbf{X} - \mathbf{W}\|_F^2 \end{aligned} \] 其中，\(\mathbf{U}\mathbf{S}\mathbf{V}^T\)是\(\mathbf{W}\)的SVD分解。

在ALM算法中，每次迭代先最小化原问题的所有变量，然后对参数\(\mathbf{Y}\)更新步长为\(\mu\)的梯度，最后更新步长\(\mu\)（一般就是乘一个大于\(1\)的数）。已有以下的结论[1]：

  • 更新过程中，当\(\mu\)有界算法保证Q-linearly收敛到最优值，当\(\mu\)无界保证super-Q-linearly收敛；
  • 最优步长已被证明为\(\mu\)；
  • 即使\(\mu\)在迭代中没有接近正无穷，算法依旧能保证收敛到最优。

在Inexact的ALM算法中，不用每次外部迭代都将\(\mathbf{A}\)和\(\mathbf{E}\)用坐标上升直到收敛，只需要分别更新一次即可（原文证明了收敛性）。 ### 2.3 收敛条件 算法收敛需要满足以下KKT条件： \[ \begin{aligned} \mathbf{D} - \mathbf{A}^* - \mathbf{E}^* = 0\\ \mathbf{Y}^* \in\partial\|\mathbf{A}^*\|_*\\ \mathbf{Y}^* \in\partial\|\lambda\mathbf{E}^*\|_1\\ \end{aligned} \] 第二项和第三项是分别对\(\mathbf{A}\)和\(\mathbf{E}\)求导得到的（这里表示\(\mathbf{Y}\)属于它们的次梯度）。实际代码来看论文中对第二和第三个条件放缩得到的式子好像有点不靠谱，用第一个式子就够了。

3. Probabilistic Principal Component Analysis(PPCA)

这一块主要是从generation和inference的角度来解释PCA。 ### 3.1 生成过程 和前面类似，数据\(\mathbf{x}\)是从latent space的向量\(\mathbf{z}\)经过一个变换矩阵\(\mathbf{W}\)的得到的。 \[ \begin{aligned} \mathop{P}(\mathbf{z})\sim\mathcal{N}(0, \mathrm{I})\\ \mathop{P}(\mathbf{d}|\mathbf{z}; \mathbf{W}, \mathbf{\mu}, \mathbf{\sigma})\sim\mathcal{N}(\mathbf{W}\mathbf{z} + \mathbf{\mu}, \mathbf{\sigma}^2\mathrm{I}) \end{aligned} \] ### 3.2 推断过程 #### 3.2.1 边际概率计算 \[ \mathop{P}(\mathbf{d};\mathbf{W}, \mathbf{\mu}, \mathbf{\sigma}) = \int\mathop{P}(\mathbf{d}|\mathbf{z}; \mathbf{W}, \mathbf{\mu}, \mathbf{\sigma})\mathop{P}(\mathbf{z})\,d\mathbf{z} \] 从\(\mathbf{x} = \mathbf{W}\mathbf{z} + \mathbf{\mu} + \mathbf{\epsilon}\)的角度来说，这是关于\(\mathbf{z}\)和\(\mathbf{\epsilon}\)的一个线性变换，因此结果仍然是一个高斯分布。主要由下面式子得到： > 假设\(\mathbf{x}\sim\mathcal{N}(\mathbf{m}_{\mathbf{x}}, \mathbf{\Sigma}_{\mathbf{x}})\)和\(\mathbf{y}\sim\mathcal{N}(\mathbf{m}_{\mathbf{y}}, \mathbf{\Sigma}_{\mathbf{y}})\)，那么有（均值和方差直接套定义算期望就可以得到） > \[ \mathbf{Ax} + \mathbf{By} + \mathbf{c}\sim\mathcal{N}(\mathbf{Am}_{\mathbf{x}} + \mathbf{Bm}_{\mathbf{y}} + \mathbf{c}, \mathbf{A}\mathbf{\Sigma}_{\mathbf{x}}\mathbf{A}^T + \mathbf{B}\mathbf{\Sigma}_{\mathbf{y}}\mathbf{B}^T) \] 因此边际概率为 \[ \mathop{P}(\mathbf{d};\mathbf{W}, \mathbf{\mu}, \mathbf{\sigma})\sim\mathcal{N}(\mathbf{\mu}, \mathbf{W}\mathbf{W}^T + \mathbf{\sigma}^2\mathrm{I}) \] #### 3.2.2 推断过程 后验概率可以直接硬解，不需要套variational inference，可以直接EM算法。 \[ \begin{aligned} P(\mathbf{z}|\mathbf{x}) = \frac{\mathop{P}(\mathbf{d}|\mathbf{z}; \mathbf{W}, \mathbf{\mu}, \mathbf{\sigma})P(\mathbf{z})}{P_{\mathbf{W}, \mathbf{\mu}, \mathbf{\sigma}}(\mathbf{x})} \end{aligned} \] 高斯分布的乘法也是一个高斯分布（除法类推），只关注上面式子的指数项： \[ \begin{aligned} &\frac{1}{\mathbf{\sigma}^2}(\mathbf{x} - \mathbf{W}\mathbf{z} - \mathbf{\mu})^T(\mathbf{x} - \mathbf{W}\mathbf{z} - \mathbf{\mu}) + \mathbf{z}^T\mathbf{z} - (\mathbf{x} - \mathbf{\mu})^T(\mathbf{W}\mathbf{W}^T + \mathbf{\sigma}^2\mathrm{I})^{-1}(\mathbf{x} - \mathbf{\mu})\\ \sim&\frac{1}{\mathbf{\sigma}^2}\{[\mathbf{W}\mathbf{z} - (\mathbf{x} - \mathbf{\mu})]^T[\mathbf{W}\mathbf{z} - (\mathbf{x} - \mathbf{\mu})] + \mathbf{\sigma}^2\mathbf{z}^T\mathbf{z}\}\\ \sim&\frac{1}{\mathbf{\sigma}^2}[\mathbf{z}^T(\mathbf{W}^T\mathbf{W} + \mathbf{\sigma}^2\mathrm{I})\mathbf{z} - (\mathbf{x} - \mathbf{\mu})^T\mathbf{W}\mathbf{z} - \mathbf{z}^T\mathbf{W}^T(\mathbf{x} - \mathbf{\mu})] \end{aligned} \] 变量代换\(\mathbf{M} = \mathbf{W}^T\mathbf{W} + \mathbf{\sigma}^2\mathrm{I}\)： \[ \begin{aligned} &\text{上式}\\ \sim&\frac{1}{\mathbf{\sigma}^2}[\mathbf{z}^T\mathbf{M}\mathbf{z} - (\mathbf{x} - \mathbf{\mu})^T\mathbf{W}\mathbf{z} - \mathbf{z}^T\mathbf{W}^T(\mathbf{x} - \mathbf{\mu})]\\ \sim&\frac{1}{\mathbf{\sigma}^2}[\mathbf{z}^T\mathbf{M}\mathbf{z} - (\mathbf{x} - \mathbf{\mu})^T\mathbf{W}\mathbf{M}^{-1}\mathbf{M}\mathbf{z} - \mathbf{z}^T\mathbf{M}\mathbf{M}^{-1}\mathbf{W}^T(\mathbf{x} - \mathbf{\mu})]\\ \sim&\frac{1}{\mathbf{\sigma}^2}[\mathbf{z} - \mathbf{M}^{-1}\mathbf{W}^T(\mathbf{x} - \mathbf{\mu})]^T\mathbf{M}[\mathbf{z} - \mathbf{M}^{-1}\mathbf{W}^T(\mathbf{x} - \mathbf{\mu})] \end{aligned} \] 观察均值和方差，得到 \[ P(\mathbf{z}|\mathbf{x})\sim\mathcal{N}(\mathbf{M}^{-1}\mathbf{W}^T(\mathbf{x} - \mathbf{\mu}), \mathbf{\sigma}^2\mathbf{M}^{-1}) \] 那么使用EM算法，每次迭代先最小化ELBO，然后将概率项换成当前的后验概率即可。 ## 4.
